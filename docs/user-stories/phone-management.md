## 1️⃣ User Story

“As a user, I want to buy a professional phone number I can use for business calls, so that I can record, transcribe, and manage calls in one place — without leaving Dialer.”

## 2️⃣ Core Features You’ll Build

| Feature                   | Description                                                   | Who Uses It     |
| ------------------------- | ------------------------------------------------------------- | --------------- |
| Search Numbers            | User searches available phone numbers by country or area code | User            |
| Purchase Number           | User selects and buys a number (via your backend → Twilio)    | User            |
| View Numbers              | User sees list of purchased numbers (active, inactive, etc.)  | User            |
| Release Number (optional) | User can release number they no longer need                   | User            |
| Assign LiveKit SIP Trunk  | System auto-links number to LiveKit room routing              | System (hidden) |
